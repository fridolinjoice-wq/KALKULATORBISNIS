# 💰 Cashflow Calculator App

Aplikasi web sederhana untuk menghitung pemasukan dan pengeluaran, serta memantau kondisi keuangan secara real-time.

---

## 🚀 Fitur

- Hitung cashflow otomatis (Pendapatan - Pengeluaran)
- Status keuangan:
  - Surplus 💚
  - Defisit ❤️
  - Seimbang 💛
- Riwayat transaksi tersimpan (SQLite)
- Grafik visual (Chart.js)
- Dark / Light mode 🌙☀️
- Reset semua data
- UI modern & responsive

---

## 🛠 Teknologi

- Python (Flask)
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js
